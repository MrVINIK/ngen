# vbomb

Установка:
apt update && apt upgrade
apt install github
apt install python
apt install python3
git clone https://github.com/MrVINIK/ngen
cd ngen 
bash setup.sh

Использование:
В начале проведите установку, затем, чтобы открывать программу
Вводить надо будет:
python main.py
(или python3 main.py)
